\doxysubsection{Detailed Description}
\begin{DoxyVerb}This class wraps the execution of OpenSPICE and convert the output to a Python data structure.

Example of usage::

  spice_server = OpenSPICEServer()
  raw_file = spice_server(spice_input)

It returns a :obj:`PySpice.Spice.RawFile` instance.\end{DoxyVerb}
